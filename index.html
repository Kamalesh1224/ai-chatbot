FAQ

What this starter is ready for

Can it answer from company documents?

Yes. It loads a default knowledge base from local text files, supports persistent admin uploads, and can still import temporary session documents through the chat widget.

Can it separate topics like HR and support?

Yes. Documents can be grouped by topic, the assistant can auto-route questions, and admins can manually focus the chat on a specific topic when needed.

Can a visitor still reach a human?

Yes. The human handoff form stays available when a visitor asks for a person or needs follow-up beyond the knowledge base.
